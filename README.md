# To-Do App

A Flutter To-Do App that allows users to manage their daily tasks efficiently. Users can **add new tasks, edit existing tasks, mark completed tasks**, and **delete tasks**. The app features a clean, interactive UI and demonstrates practical Flutter skills including state management and navigation.

---

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Interactive and clean UI
- Beginner-friendly Flutter project
